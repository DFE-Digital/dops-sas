Installation

This is a beta product.

All known issues and bugs can be found on the GitHub repo (opens in new tab) for this service.

If you decide to use this product in your organisation, please fork, and contribute changes into the main repository.

If you want to contribute to this product, you may want to check our backlog for features we have not yet implemented and are looking for help with.

Expected setup timescales

If you follow these instructions step by step, it should take around 1.5 hours from starting, to using the service.

Prerequisites

This is a beta product which we are testing the suitability for supporting the service assessment process in a government department, we are using low-cost hosting and database provision to keep costs at a minimum.

Hosting

This service is hosted using Heroku. It is sufficient for the intermittent, non-continuous use of the product.

We use a standard eco dyno which is about Â£6 a month.

Database

We use a Postgres database which is provided as an add-on in Heroku.

We are using a Free tier instance which is sufficient for supporting this product.

Notify

We use GOV.UK Notify to send emails from this service.

All templates and parameters are detailed in the GitHub repo in the email_templates folder.

This is free for government use.

Codebase

We chose to build this product using Node.JS. This enables the product to be built and contributed to, by a wide range of people, who might not normally contribute code had we built it using Ruby or C#.

This also enables us to build quickly, make use of common components and patterns more easily, and be supported by a team familiar with the tech stack used.
